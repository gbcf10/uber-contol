# 🚗 Uber Finance Control

Sistema para controle financeiro diário de motoristas de aplicativo.

## 📌 Funcionalidades

- Registro de ganhos e gastos
- Cálculo automático de lucro
- Relatório semanal (segunda a domingo)
- Relatório mensal
- Controle de dias de folga

## 🛠 Tecnologias

- Node.js
- Express
- JSON (simulando banco de dados)

```bash
npm install
node server.js"# uber-contol"
